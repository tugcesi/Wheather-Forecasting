# Weather Forecasting Application

## Project Description
This weather forecasting application provides accurate weather predictions for different locations. It retrieves data from reliable weather APIs and presents it in a user-friendly format.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/tugcesi/Wheather-Forecasting.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Wheather-Forecasting
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage Information
To start using the application:
1. Run the following command:
   ```bash
   npm start
   ```
2. Open your web browser and go to `http://localhost:3000` to access the application.
